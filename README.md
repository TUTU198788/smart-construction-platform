# 智建云链 - 智能建造与产业链协同创新平台

这是一个基于 Web 的智能建造与产业链协同创新平台，提供以下主要功能：

- AI 方案设计
- AI 联合设计中心
- 数智工厂
- 工业化建造
- 工程保险平台
- 产业人才平台
- 创新技术转化
- 专利协作与维权平台

## 技术栈

- HTML5
- CSS3
- JavaScript
- Three.js (用于 3D 效果)
- Font Awesome (图标库)
